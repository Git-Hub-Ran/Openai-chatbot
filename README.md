# OpenAI Chatbot

This project is a simple command-line chatbot built with Python and the OpenAI API.

The chatbot sends user input to OpenAI and returns a generated response.

---

## Requirements

- Python 3.9+
- An OpenAI API key

---

## Installation

1. Clone or download this repository.

2. Install the required Python packages:

```bash
pip install -r requirements.txt
